# Polla Mundial 2026 ⚽🏆

Aplicación web para la Polla del Mundial de Fútbol USA/CAN/MEX 2026.  
**44 años de tradición (1982-2026) — 12ª edición**

## Ejecutar localmente

```bash
pip install -r requirements.txt
python app.py
```

Abre http://localhost:5000

**Admin por defecto:** `admin` / `admin2026`

## Desplegar en Render (gratis)

1. Sube el proyecto a un repositorio de GitHub
2. Ve a [render.com](https://render.com) y crea una cuenta
3. Clic en "New > Web Service"
4. Conecta tu repositorio
5. Render detectará automáticamente la configuración
